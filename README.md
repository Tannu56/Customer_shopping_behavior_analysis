# Customer Shopping Behavior Analysis

## Overview

This project focuses on analyzing customer shopping behavior using Data Analytics techniques. The objective is to extract meaningful insights from customer data, identify purchasing patterns, and support data-driven business decisions. The project covers the complete analytics workflow, from data preparation to visualization and reporting.

## Dataset

The dataset contains customer shopping information, including demographic details, purchase history, product categories, payment methods, and shopping preferences.

## Tools & Technologies

* Python (Pandas)
* PostgreSQL
* SQL
* Power BI
* Microsoft PowerPoint

## Project Steps

### 1. Data Loading

* Imported the dataset using Python.
* Performed initial data inspection and validation.

### 2. Exploratory Data Analysis (EDA)

* Analyzed customer demographics and purchasing behavior.
* Identified trends, distributions, and key patterns.
* Created visualizations to understand the data.

### 3. Data Cleaning

* Handled missing values and duplicate records.
* Corrected data inconsistencies.
* Prepared the dataset for further analysis.

### 4. SQL Analysis

* Loaded the cleaned dataset into PostgreSQL.
* Executed SQL queries to answer business-related questions.
* Generated insights on customer behavior and sales trends.

### 5. Dashboard Development

* Built an interactive Power BI dashboard.
* Visualized key metrics and business insights.
* Enabled easy exploration of customer shopping patterns.

### 6. Reporting & Presentation

* Created a detailed project report summarizing findings.
* Designed a PowerPoint presentation to communicate results effectively.

## Dashboard Highlights

* Customer Demographics Analysis
* Purchase Behavior Insights
* Product Category Performance
* Payment Method Distribution
* Key Business KPIs

## Results

The analysis revealed valuable insights into customer preferences, purchasing trends, and product performance. These findings can help businesses improve marketing strategies, customer engagement, and overall decision-making.

## How to Run

1. Clone the repository.
2. Install required Python libraries.
3. Run the data cleaning and EDA notebooks/scripts.
4. Import the cleaned dataset into PostgreSQL.
5. Execute the SQL queries provided in the project.
6. Open the Power BI dashboard file to explore visual insights.
7. Review the project report and presentation for detailed findings.
